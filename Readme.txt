О программе:

- Програма скачивает HTML-страницу посредством HTTP-запроса, сохраняет загруженный файл на жесткий диск компьютера, подсчитывает количество уникальных слов на странице и выводить статистику по словам в консоль. Программа имеет возможность сохранение результата в базу данных.


Системные требования:

- PC совместимый компьютер;
- Версия ОС: Windows 7, 8, 10;
- .NET Framefork 4.x;


Инструкция по установке и запуску:

- Программа не требует установки, запускается посредством запуска файла TestTask.exe


Руководство пользователя:

- Запустите файл TestTask.exe, после "http:\\" введите адрес русскоязычного сайта например www.simbirsoft.com и нажмите на клавишу <ENTER>. После обработки информации в консоле выведется статистика по частоте вхождения уникальных слов на интернет странице. Далее выводится запрос на сохранение результата в базе данных. Для продолжения работы с программой нажмите на <ENTER> или введите любое слово и нажмите на <ENTER>, для выхода из программы в ответ на вопрос о выходе из программы введите слово "да" и нажмите на <ENTER>.
